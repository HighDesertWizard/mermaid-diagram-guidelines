# Changelog

All notable changes to the Mermaid Diagram Guidelines (MDG) are documented here.

## [1.0] — 2026-03-07

**Public release.** Single-file consolidation of all MDG content into `SKILL.md`. Four rounds of structured multi-LLM peer review with five reviewer models (GPT, Gemini, Grok, Claude Sonnet, Claude Haiku). No breaking changes from v0.49.

### What's included
- Emergency Rules (10 critical syntax and visibility rules)
- Layout Techniques T1–T6 (transparent wrapper, side-by-side, stacked, aspect ratio, grid, hub-and-spoke)
- Page-Fit Strategy (cross-grain direction + subgraph-level connections)
- 8 diagram type references (flowchart, sequenceDiagram, erDiagram, stateDiagram-v2, classDiagram, gantt, mindmap, pie)
- Color palettes (standard, subgraph backgrounds, color-blind friendly)
- Advanced layout techniques (invisible elements, width control, routing artifact mitigation)
- RAG Mode for knowledge-graph extraction
- Pre-publish checklist
- Layout troubleshooting guide
- ELK layout engine reference
